## Security Reporting

If you discover a security vulnerability, unsafe behavior, or a potential abuse vector in ViNNi:

- **Do NOT** open a public GitHub issue
- **Do NOT** disclose the issue publicly before coordination

Instead, responsibly report the issue directly to the creator.

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Potential impact
- Any suggested mitigations

Responsible disclosure helps keep ViNNi safe for everyone.
